## Projeto: Análise de Vendas – Superstore

Este projeto apresenta uma análise de vendas de uma empresa varejista fictícia utilizando Power BI.

### KPIs
- Faturamento
- Lucro
- Ticket Médio
- Quantidade Vendida

### Ferramentas
- Power BI
- Power Query
- DAX

### Principais insights
- A região West apresenta maior faturamento
- Technology é a categoria mais lucrativa
- Existe sazonalidade positiva no último trimestre
