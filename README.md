# 📊 Projetos

## 1️⃣ Olist Sales Performance Dashboard

Dashboard desenvolvido para analisar o desempenho de vendas de um marketplace brasileiro utilizando o dataset público da Olist.

**Principais análises:**

- Faturamento mensal
- Ticket médio
- Total de clientes
- Top cidades e estados por faturamento
- Faturamento por categoria
- Métodos de pagamento mais utilizados

---

## 2️⃣ Análise de Vendas – Superstore

Dashboard de análise de vendas utilizando o dataset público Superstore.

**Principais análises:**

- Faturamento total
- Lucro
- Ticket médio
- Desempenho por região
- Desempenho por categoria de produto
