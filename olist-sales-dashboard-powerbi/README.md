# 📊 Olist Sales Performance Dashboard

Este projeto apresenta um **dashboard de Business Intelligence desenvolvido no Power BI** com o objetivo de analisar o desempenho de vendas de um marketplace brasileiro utilizando o dataset público da Olist.

O objetivo é transformar dados brutos em **insights visuais que apoiem a tomada de decisão**, explorando tendências de receita, comportamento de clientes e distribuição geográfica das vendas.

---

# 🎯 Objetivo do Projeto

Responder perguntas de negócio relevantes para um e-commerce, como:

- Como o faturamento evolui ao longo do tempo?
- Quais cidades e estados concentram maior volume de vendas?
- Quais categorias de produtos geram mais receita?
- Quais métodos de pagamento são mais utilizados pelos clientes?

---

# 📈 KPIs Principais

O dashboard apresenta os seguintes indicadores-chave de performance:

- **Faturamento Total**
- **Quantidade de Pedidos**
- **Ticket Médio**
- **Total de Clientes**

---

# 📊 Visualizações do Dashboard

O dashboard inclui as seguintes análises:

- **Faturamento Mensal** – análise de tendência de receita ao longo do tempo  
- **Top Cidades por Faturamento**  
- **Top Estados por Faturamento**  
- **Faturamento por Categoria de Produto**  
- **Métodos de Pagamento Mais Utilizados**

Essas visualizações permitem analisar o desempenho do marketplace sob diferentes perspectivas, incluindo **tempo, localização geográfica e comportamento de compra**.

---

# 🛠️ Ferramentas Utilizadas

- **Power BI**
- **Power Query** para transformação de dados
- **Modelagem de dados**
- **DAX para criação de métricas**

---

# 📂 Dataset

O projeto utiliza o **Brazilian E-commerce Public Dataset by Olist**, disponível no Kaggle.

Este dataset contém informações sobre pedidos, clientes, produtos, pagamentos e avaliações de compras realizadas em um marketplace brasileiro.
